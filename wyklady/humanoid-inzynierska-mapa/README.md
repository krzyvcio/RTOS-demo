## TL;DR
System humanoida łączy grafy kinematyczne, kontaktowe, sieciowe, stanów i błędów z matematyką:
algebra liniowa, grupy Liego, ODE, teoria sterowania, optymalizacja wypukła/nieliniowa,
probabilistyka, teoria harmonogramowania i analiza sygnałów.
