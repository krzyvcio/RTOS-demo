# Digital Twin Lab dla Systemu Wirującego (Neutralny Kontekst) — Wykłady

Cel tego folderu: zebrać komplet narzędzi (IT + HW) i metodykę, aby **wirtualnie testować ogólny system wirujący** zanim powstanie hardware:
- model ODE/MDOF,
- sterowanie i filtracja,
- czas rzeczywisty (Linux PREEMPT_RT / RTOS),
- symulacja magistral (EtherCAT/CAN),
- HIL (MCU w pętli),
- analiza sygnałów,
- pipeline DevOps/CI.

Uwaga: materiał jest neutralny aplikacyjnie i skupia się na praktyce inżynierskiej.
